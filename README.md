# Learning Go

Notes from [Todd McLeod's Udemy course](https://www.udemy.com/course/learn-how-to-code/) | [Course repo](https://github.com/GoesToEleven/learn-to-code-go-version-03)

## Topics Covered

| # | Topic | Key Concepts |
|---|-------|--------------|
| 003 | Basics | comments, packages, imports, `fmt.Printf` verbs, raw strings |
| 013-014 | Variables | `:=`, `var`, blank identifier `_`, `%b` `%x` formatting |
| 017-018 | Types | type conversion, functions, multiple returns, zero values |
| 024-026 | Constants | bit shifting `<<`, `iota`, custom types (ByteSize) |
| 027-030 | Exercises | variable declaration patterns, print verbs, numeric limits |
| 033 | Scope | package/block scope, variable shadowing, methods |
| 049-054 | Modules | external packages, go.mod, dependency versions, git tags |
| 066-070 | Control Flow | `if`/`else`, logical operators, comma-ok idiom, `switch`, `select` |
| 071 | Concurrency | goroutines, channels, concurrency vs parallelism |
| 072-075 | Loops | for (3 forms), `break`, `continue`, nested loops, `range`, modulus |
| 076-092 | Exercises | control flow, loops, slices, maps, comma-ok idiom |
| 096-097 | Arrays | fixed-size, `[...]` literals, `len()` |
| 098-105 | Slices | dynamic sizing, `append`, slicing `[i:j]`, `make`, delete, 2D slices |

## Running Examples

```bash
go run <folder>/main.go
```
